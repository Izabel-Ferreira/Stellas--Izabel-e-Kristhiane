# Stellas--Izabel-e-Kristhiane
Identificamos a necessidade de criar uma cidade mais sustentável, capaz de equilibrar desenvolvimento urbano, qualidade de vida e preservação do meio ambiente. Observamos desafios como o consumo excessivo de energia, a geração de resíduos e a necessidade de meios de transporte mais eficientes e menos poluentes.Desenvolvemos, no Delightex, um cenário virtual representando uma cidade sustentável. O projeto inclui elementos como áreas verdes, fontes de energia renovável, sistemas de reciclagem, espaços públicos planejados e opções de mobilidade sustentável, demonstrando como uma cidade pode funcionar de forma mais consciente e eficiente.Utilizamos as ferramentas de modelagem e simulação do Delightex para construir os ambientes da cidade. Pesquisamos conceitos de sustentabilidade urbana, selecionamos os elementos mais relevantes e os organizamos em um cenário interativo. Durante o desenvolvimento, buscamos representar soluções reais que podem contribuir para a redução dos impactos ambientais.O projeto resultou em uma representação visual e interativa de uma cidade sustentável, facilitando a compreensão dos conceitos estudados. Além de desenvolver habilidades de planejamento e criatividade, o trabalho permitiu refletir sobre a importância da sustentabilidade e mostrou como a tecnologia pode ser utilizada para apresentar soluções para os desafios urbanos do futuro.
